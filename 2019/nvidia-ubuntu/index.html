在 Ubuntu 18.04 上安装 NVIDIA 有三种方法：

  • 使用标准 Ubuntu 仓库进行自动化安装
  • 使用 PPA 仓库进行自动化安装
  • 使用官方的 NVIDIA 驱动进行手动安装
