使用 V2Ray 翻越防火长城

2020.11.15 更新

以下教程针对旧版 VMESS 协议，新版 VLESS 协议请按照配置内容和软件界面自行摸索。

2020.04.29 更新

更新 Trojan 的搭建和使用，可与 v2ray 共存：https://morooi.cn/2020/trojan

各平台详细配置过程

  • MacOS
  • Windows
  • Chrome + SwitchyOmega (推荐)
  • iOS / iPad OS
  • Android

MacOS

下载客户端：V2RayU: [https://github.com/yanue/V2rayU/releases]

导入配置

先复制配置链接，然后选择从粘贴板导入

至此配置导入完成

…

启用代理

点击 Trun v2ray-core On，并确定模式为 Pac 模式，开启成功。

打开 Google: https://www.google.com

几种模式解释：

  • Pac 模式：匹配规则，规则内网站走代理，其他直连（规则可在下面 Pac 设置中修改或更新）
  • 全局模式：所有网站走代理（这里全局仅仅是针对浏览网页）
  • 手动模式：直连模式，所有网站不走代理

Windows

下载客户端：V2RayN: [https://github.com/2dust/v2rayN/releases]（仅下载 `v2rayN-Core.zip`）

解压后打开 v2rayN.exe

导入配置

先复制配置链接

在状态栏找到图标，右键选择从剪切板导入批量 URL，随即弹出导入成功提示框

启动代理

同样右击图标，选择启用 Http 代理，然后在 Http 代理模式中选择开启 PAC, 并自动配置 PAC (PAC 模式)

开启成功

打开 Google: https://www.google.com

几种模式解释：

  • 全局模式：所有网站走代理（这里全局仅仅是针对浏览网页）
  • PAC 模式：匹配规则，规则内网站走代理，其他直连
  • 直连模式：所有网站不走代理

Chrome 配合 SwitchyOmega (推荐)

Chrome 配合 SwitchyOmega 使用更加灵活，如果使用 PAC 模式，有些网站可能不在规则列表里，导致访问巨慢，又需要切换到全局模式才能打开，切来切去很是麻烦。

借助 SwitchyOmega，可以一键将规则列表外的网站添加到自定义规则里，同时不影响国内网站的正常访问

需要使用 Chrome 或 Edge (Chromium 内核版)

安装 SwitchyOmega：Proxy SwitchyOmega - Chrome 网上应用店

这里准备好了一份基本配置文件，可以直接导入

打开 SwitchyOmega，选择 导入/导出，在 在线恢复 一栏中填入如下地址：https://morooi.cn/2019/v2ray/OmegaOptions.bak，点击 恢复

打开 v2rayU /v2rayN 的设置，找到图示设置，记住该端口号

在 SwitchyOmega 的 新世界 中，确保端口号和上一步的一致

打开 v2rayU (macOS) 的手动模式，或 v2rayN (Windows) 的直连模式，并启动，保持后台运行

在 SwitchyOmega 中选择 自动选择 即可按照规则自动配置爬墙

在 自动选择 设置中，可以手动添加网站选择是否代理，本配置文件已经添加了 GitHub 的规则

更多使用说明可查看 SwitchyOmega 项目 Wiki：https://github.com/FelisCatus/SwitchyOmega/wiki

iOS / iPad OS

下载客户端：App Store 使用非国区账号搜索安装 Quantumult X 或 Shadowrocket（均需付费购买）

Quantumult X 的使用指南：Quantumult X 不完全教程

使用共享账号千万别登陆 iCloud，仅登陆 App Store！！否则手机可能被锁定且无法恢复

分享一些规则：

  • 神机规则
  • Clash / Loon / QuantumultX / Shadowrocket / Surge 去广告规则
  • Shadowrocket 配置

Android

下载客户端：BifrostV: [https://pan.baidu.com/s/1VuuUeImkpcrLUwvG76RJqQ] （提取码：hufi）

打开应用，点击右上角，添加规则图标

选择从 URL 导入，输入配置 URL，即导入成功

选中配置文件，点击右下角 Logo，开启成功
