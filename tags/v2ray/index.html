使用条件

本文写于 2022.12.3，后续软件可能有更新

  • macOS 平台暂无软件支持，需安装 Xray 二进制程序手动运行，然后通过 v2rayU、clashX、clashX Pro 等代理软件连接
  • iOS /iPadOS 平台暂无软件支持，可使用具有公网 IP 的服务器搭建任意代理服务（如 Shadowsocks）转发，通过 Quantumult X、ShadowRocket 等代理软件连接中转机
  • Windows 平台使用 v2rayN v5.38 及后续版本
  • Android 平台使用 v2rayNG v1.7.27 及后续版本

注：实际所有平台都可使用中转机的方式实现，但需一层中转且消耗服务器流量，而且服务器通常是实名开通，只能作为过渡方案。
